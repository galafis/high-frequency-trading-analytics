# Data Directory

This directory holds datasets used by the high-frequency trading analytics system.

## Structure

```
data/
├── raw/
│   └── sample_data.csv       # Sample market data with timestamp and price columns
└── processed/
    ├── features.parquet       # Engineered features in Parquet format
    └── features.csv           # Engineered features in CSV format
```

## Raw Data

`raw/sample_data.csv` contains sample time-series market data used as input to the feature engineering pipeline. It must include at least the columns `timestamp` and `price`.

## Processed Data

Files in `processed/` are generated by running the feature engineering pipeline:

```bash
python src/data/features.py --input data/raw/sample_data.csv --output_parquet data/processed/features.parquet --output_csv data/processed/features.csv
```

## Supported Formats

- **CSV** — Tabular data, easy to inspect
- **Parquet** — Columnar format, more efficient for analytical workloads

## Notes

- Large or sensitive data files should not be committed to the repository.
- Use `.gitignore` to exclude any files that shouldn't be tracked.
