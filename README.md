## Overview | Visão Geral

### 🇺🇸 English

A modular high-frequency trading (HFT) analytics platform for researching, developing, and evaluating trading algorithms. It covers quantitative analysis, market microstructure, and feature engineering for automated trading strategies.

### 🇧🇷 Português

Plataforma modular de análise de trading de alta frequência (HFT) para pesquisa, desenvolvimento e avaliação de algoritmos de negociação. Abrange análise quantitativa, microestrutura de mercado e engenharia de features para estratégias de trading automatizadas.

## ✨ Main Functionalities | Funcionalidades Principais

- **Feature Engineering**: Pipeline to compute common market features for HFT — lags, returns, moving averages, volatility, and order book imbalance (OBI).
- **Data Validation**: Tools that verify the schema and integrity of input CSV data.
- **Backtesting Engine**: Module for simulating and evaluating trading strategy performance against historical data.
- **Interactive Dashboard**: A Streamlit application for visualizing raw data and engineered features.
- **Modular Structure**: Well-organized code structured for straightforward maintenance and extension.

## 🚀 Quick Start | Início Rápido

### Prerequisites

Ensure you have Python 3.8+ installed. Using a virtual environment is highly recommended.

```bash
# 1. Clone the repository
git clone https://github.com/galafis/high-frequency-trading-analytics.git
cd high-frequency-trading-analytics

# 2. Create and activate a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`

# 3. Install dependencies
pip install --upgrade pip
pip install -r requirements.txt
```

### Running the Platform

```bash
# 1. Generate sample data (optional, sample data is already included)
# The script used to generate the current sample_data.csv is available for reference.

# 2. Run the feature engineering pipeline
python src/data/features.py --input data/raw/sample_data.csv --output_parquet data/processed/features.parquet --output_csv data/processed/features.csv

# 3. Run the interactive dashboard
streamlit run src/dashboard.py

# 4. Run tests to ensure everything is working correctly
pytest
```
